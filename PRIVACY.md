# Privacy Policy - Context-Awared Text Expander

**Last Updated:** January 2025

## 1. Information We Collect

Context-Awared Text Expander collects the following information to provide its services:

### 1.1 Shortcut Data
- Shortcut abbreviations and expansions
- Category assignments
- Usage statistics and timestamps
- Custom variables

### 1.2 User Information
- User email address (for document access)
- User preferences and settings

### 1.3 Configuration Data
- Template definitions
- Custom variable values
- Sync settings

### 1.4 Activity Logs
- Timestamps of shortcut usage
- Template applications
- Configuration changes

## 2. How We Use Your Information

We use your information to:
- Expand text shortcuts in documents
- Analyze document context for smart expansions
- Track usage patterns
- Sync shortcuts across documents

## 3. Data Storage and Retention

### 3.1 Storage Location
- All configuration data is stored in Google Apps Script PropertiesService
- Shortcut data is stored in user properties
- No external servers or databases

### 3.2 Data Retention
- Configuration data is retained until you delete it
- Activity logs are limited to the most recent 100 entries
- Shortcut data remains until you delete it

## 4. Data Sharing

We do not sell, rent, or share your data with third parties for marketing purposes.

### 4.1 Limited Sharing
- No data sharing occurs
- All data remains within your Google account

## 5. Data Security

We implement appropriate security measures:
- All data is stored within Google's secure infrastructure
- OAuth 2.0 authentication for Google API access
- Document access is limited to your documents
- No external servers or databases

## 6. Your Rights

You have the right to:
- Access your shortcut data and settings
- Delete shortcuts and configuration data
- Clear activity logs
- Uninstall the add-on at any time

## 7. Children's Privacy

Context-Awared Text Expander is not directed to children under 13. We do not knowingly collect personal information from children under 13.

## 8. Changes to This Policy

We may update this privacy policy from time to time. We will notify you of any changes by posting the new policy on this page.
